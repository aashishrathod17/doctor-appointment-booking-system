# 🏥 Doctor Appointment Booking System

This project is a **Doctor Appointment Booking System** developed as a web-based application that allows users to easily select their health problem, choose a doctor, and book an appointment online.

The system follows a structured user flow to provide a smooth and user-friendly booking experience.

---

## 👨‍💻 About the Project

The main objective of this project is to simplify the process of booking doctor appointments digitally.

Users can select their health issue, view relevant doctors, and book appointments in just a few steps.

This project demonstrates my skills in **frontend web development and UI/UX flow design**.

---

## 🔄 Project Workflow

The application follows a clear step-by-step booking process:

START
→ Home Page (index.html)
→ Click "Select Your Health Problem"
→ Health Problem Selection Page (AR_booking.html)
→ Select Health Category (Heart / Skin / Eye)
→ Doctor Listing Page
→ Select Doctor
→ Appointment Booking Form
→ Fill Details (Name, Mobile, Date, Time, Problem)
→ Submit Form
→ Appointment Confirmation Display
→ END

This structured flow ensures an intuitive and seamless user experience.
